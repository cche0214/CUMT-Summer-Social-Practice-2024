2024年高等学校实验室安全管理培训班在徐举办

发布者：陈岳豪 发布时间：2023-12-27

笑脸

7月17日至20日，由江苏省高等教育学会高校实验室研究委员会主办，我校承办的2024年高等学校实验室安全管理培训班在徐州召开，来自全省70余所高校600余人参加了本次培训班。

我校副校长王忠宾出席开班仪式并致辞。他介绍了我校办学历史及发展情况，并指出，高校实验室是人才培养、科学研究的重要场所，是“双一流”建设的重要支撑。实验室安全直接关系到广大教师和学生的生命财产安全，关系学校和社会的安全稳定。当前高校实验室安全工作总体趋于向好，但是形势依然严峻，存量风险不断集聚，增量风险不断涌现，对实验室安全管理工作提出了更高要求。通过本次培训，必将进一步提升我省实验室安全管理工作人员的安全意识、知识、技能水平，推动我省高校实验室安全工作迈向更高水平。

江苏省教育厅科技产业处徐宁致辞，并就2024年江苏地方高校实验室安全现场检查安排作了说明。他指出，高校实验室安全是国家总体安全的重要组成部分，事关教育事业改革发展全局和广大师生切身利益。实验室安全管理人员要准确把握习近平总书记关于安全生产工作的新理念、新思想的科学内涵和精神实质，深刻汲取高校实验室安全事故惨痛教训，不断增强发展的安全性。

南京大学校长助理李斌代表江苏省高校实验室研究会致辞，他指出，本次培训班是贯彻落实教育部和省教育厅关于加强高校实验室安全管理工作具体要求，总结今年上半年全省高校实验室管理安全管理自查自纠和现场检查情况，提升我省高校实验室安全管理工作能力和水平的重要举措。相信本次培训班的顺利开办，必将深入推进我省高校实验室安全管理工作再上新台阶。

中国高等教育学会实验室管理研究会秘书长刘克新教授代表中国高等教育学会实验室管理工作分会对本次培训班的召开表示祝贺，他指出，江苏省高校实验室安全管理工作一直走在全国前列，希望广大实验室安全管理人员通过本次培训，共同树立有担当、讲科学、低调务实的工作理念，在实验室安全管理工作岗位上做出优异成绩。
